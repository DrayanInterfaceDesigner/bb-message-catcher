"index.html"
====================================


A ideia inicial era apagar mensagens que poluíam visualmente a página após muito tempo, o que me atrapalhava para prestar
atenção nas aulas.
A ideia agora é reaproveitar o código para coletar as mensagens, salvando-as em JSON(CORS Policy-JS não permite, mas dou um jeito)
ou enviando por conversação com uma interface em C#, para que meus professores possam usar seus programas que ficam em tela 
cheia, sem precisarem se preocupar com minimizar os softwares toda vez apenas para verem uma mensagem.

Ferramenta criada para o site BLACKBOARD, plataforma privada da microsoft.
A página salva "Tema_ O que é ser um universitário_ O pensamento crítico e a pesquisa acadêmica_ o porquê da filosofia na educação. - Bb Collaborate"
(depois renomeada para "index.html), é usada apenas como exemplo.

O app já funciona em aulas ao vivo e se comporta dinâmicamente, atualizando toda vez que uma mensagem é enviada.

InitEraser("Name") // inicia a aplicação
EraseHer("Name") // apenas apaga as mensagens já dispostas, não inicia a aplicação.

**Para usar, utilize o console, pois a função não é chamada ao carregar :)
